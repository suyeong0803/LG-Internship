# 요구사항 명세서
## 1. 프로젝트 개요
녹화영상을 이용하여 숏폼 컨텐츠를 제작하고 영상처리 기술을 경험한다.



<br>



## 2. 기능적 요구사항
<table>
<th> 기능</th>
<th colspan="2"> 내용</th>
<tr>
<td rowspan="3"> 불러오기 </td>
<td> 기본동작</td>
<td> 1. 불러오기 버튼을 누른다. <br> 2. 파일 탐색기에서 동영상 파일을 선택한다. <br> 3. 동영상 파일이 플레이리스트에 올라간다. </td>
</tr>
<tr>
<td> 대안 동작</td>

</tr>
<tr>

<br>2. 잘못된 경로일 경우 "동영상을 찾을 수 없습니다. 파일 경로를 확인해주세요." 알림 메세지를 띄운다.
